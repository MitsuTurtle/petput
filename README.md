## :turtle:概要
ペット写真共有SNSです。

## :turtle:制作背景
私には25年以上飼育しているカメがいます。<br>
表情や仕草で一緒に生活している私たちに癒やしを与えてくれます。

<img src="https://i.imgur.com/uWiMge8.jpg" width="250px"><br>
そんなカメについて
* 記録を残してあげたい
* かわいさを伝えたい
* 他のカメの写真も見てみたい
* 他の飼い主さんと交流したい

との思いがあります。<br>
ペットの飼い主さんには、きっと、同じ思いをもっている人がいるのではと思い、このたび&ensp;**PetPut**&ensp;を作成することとしました。
